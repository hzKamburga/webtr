# ⚡ WebTR Resmi Paketleri

Burası WebTR çekirdek ekibi tarafından bakımı yapılan resmi paketlerin bulunduğu alandır.

## Paketler

| Paket | Tür | Açıklama |
|-------|-----|----------|
| **@glass** | Stil | Modern cam efektli UI elementleri |
| **@ui** | Stil | Temel buton, input ve kart stilleri |
| **@player** | Lib | Video oynatıcı mantığı |
| **@player-ui** | Stil | Video oynatıcılar için modern temalar |

## Katkı

Resmi paketlere katkıda bulunmak için Pull Request gönderebilirsiniz.
